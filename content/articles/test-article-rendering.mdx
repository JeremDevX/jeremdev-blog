## Getting Started with CSS Visual Effects

CSS has evolved far beyond simple color and font changes. Modern CSS enables stunning visual effects that were once only possible with JavaScript or image editors.

### Box Shadows and Depth

One of the most versatile CSS properties for creating depth is `box-shadow`. Here's how to create layered shadows for a realistic effect:

```typescript
interface ShadowConfig {
  offsetX: number;
  offsetY: number;
  blur: number;
  spread: number;
  color: string;
}

function generateShadow(configs: ShadowConfig[]): string {
  return configs
    .map(({ offsetX, offsetY, blur, spread, color }) =>
      `${offsetX}px ${offsetY}px ${blur}px ${spread}px ${color}`
    )
    .join(", ");
}

const layeredShadow = generateShadow([
  { offsetX: 0, offsetY: 2, blur: 4, spread: 0, color: "rgba(0,0,0,0.1)" },
  { offsetX: 0, offsetY: 8, blur: 16, spread: 0, color: "rgba(0,0,0,0.1)" },
]);

console.log(layeredShadow);
```

### Key Techniques

Here are some essential techniques for visual effects:

- **Gradients** — Linear and radial gradients for smooth color transitions
- **Filters** — Blur, brightness, contrast, and more
- **Transforms** — Scale, rotate, and skew elements
- **Animations** — Keyframe-based motion and transitions
